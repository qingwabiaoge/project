#exec(cb)
使用链式语句时，可以在之后接.exec()执行查询，并指定回调函数。

```
blogModel.find({title: "Mongoose", meta.votes: 100}, {title: 1, author: 1, body: 1})
 .exec(function(err, docs){
if(err) console.log(err);
console.log('查询结果：' + docs);
})
```
