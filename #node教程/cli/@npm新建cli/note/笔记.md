

# npm是什么?

>npm是node自带的一个插件,

>通过npm上传代码到npm仓库 

>通过npm下载安装线上npm仓库的代码到本地





# 版本

    npm -v

# 升级

    npm install npm@latest -g

# 创建模块和package.json

    npm init
    npm init -y 默认

# 全局安装和局部安装

###  全局安装和运行

```
npm i nodemon -g

```
全局运行的命令

```
命令行  nodemon xxx即可运行
```
```
webpack生成的命令
```

```
npm run xxx
```



### 局部安装和运行

```
npm i nuxt

npm i nuxt -S //安装并保存到webpack.json生产环境配置

npm i nuxt -d //安装并保存到webpack.json开发环境配置


npm i xxx@latest  //安装最后一个版本

```

```
npm run xxx

```
