

# npm是什么?

>npm是node自带的一个插件,

>通过npm上传代码到npm仓库 

>通过npm下载安装线上npm仓库的代码到本地


# 安装npm


##### 版本

    npm -v

##### 升级

    npm install npm@latest -g


##### 配置npm的path
![1](./2.png)


# 创建模块和package.json

    npm init
    npm init -y 默认



# 通过npm安装模块和运行模块

###  全局安装和运行cli模块

全局安装cli模块

```
npm i nodemon -g

```
全局运行的命令

```
命令行  nodemon xxx即可运行
```
```
webstorm生成的命令
```
```
   npm 间接运行
    "dev": "node 1.js",
    "dev1":"babel-node",
    "dev2": "set NODE_ENV=production"  
     npm run dev
```



### 局部安装和运行cli模块


局部安装cli模块
```
npm i nuxt

npm i nuxt -S //安装并保存到webpack.json生产环境配置

npm i nuxt -d //安装并保存到webpack.json开发环境配置


npm i xxx@latest  //安装最后一个版本

```
局部运行cli模块
```
npm 间接运行
 "dev": "cross-env NUXT_HOST=0.0.0.0 NUXT_PORT=3333 nuxt",
"dev2": "nuxt"
 npm run dev

```

### 局部安装和引入对象模块
```
import axios from 'axios'//npm会自动搜索

```
