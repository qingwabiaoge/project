# el对象和vue对象

dom    |vuedom   |  生成对象

标签   |自定义的标签|对象构造函数运行式
标签的属性 |自定义标签的属性|构造函数的属性   

```html
<div id="app" data-src="www">
    <h1></h1>
</div>

<script>
    const obj = document.getElementById('app')
    console.log({obj})

</script>

```
