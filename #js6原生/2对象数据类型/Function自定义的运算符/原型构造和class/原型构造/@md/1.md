# 函数和对象的关系

>Object Array等构造器是函数,函数创造对象
fn instaceof Objcet===true,所以函数实例fn是对象

### 函数辅助构造对象

##### {a:1,b:2}

{a:1,b:2}等价于new object({a:1,b:2}) ,建立对象没有Fn构造函数的辅助


##### new Fn(i,m)

new Fn(i,m)等价于Fn(i,m).call(new Objcet())

有构造函数辅助生成对象

可以设置函数参数
