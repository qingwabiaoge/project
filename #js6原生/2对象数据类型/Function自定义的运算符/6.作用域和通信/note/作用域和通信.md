```js

```

###### 1. 通过vuex通信,$store的地址注入到每个组件里, 组件通过this.$store通信   , vue监听$store变化,{{$store.state.xxx}}更新到视图
![](vuex.png)


```js
   let store = {
        state: undefined,
        setState(i) {
            this.state = i
        }
    }

    class Vm {
        constructor(a, b) {
            this.a = a;
            this.b = b;
            this.$store = store
        }
    }

    let vm = new Vm(1, 2)
    vm.$store.setState(3)

    let vm2 = new Vm(10, 20)
    vm2.$store.setState(30)

```
