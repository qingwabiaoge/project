




# 封装vue组件构造器(类的复用)



##### 解藕:
组件化化是如表格和表单本来可以写在一起分开,tab和tab-pane本来可以写在一起的分开,

解藕的作用模块尽量小,越小的越方便复用

#####  内聚:
模块化组件化标签化:__把一个逻辑的dom  css  js逻辑  数据状态 封装在一起,__简化__成标签,


### 父子通信


##### 自定义prop 和@event


```
原理父组件对象的变量和函数作为子组件构造器的参数,构造函数运行生成子对象
```
```
 父子组件通信映射到视图层

父data->子prop(cl.prop)->vue指令->视图层

父data变化->子视图变化

```

```
//fa.js伪代码
```
