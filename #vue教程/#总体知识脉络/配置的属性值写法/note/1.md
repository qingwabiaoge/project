
# vue配置文件对象和原生对象的属性值

### 基本值做对象的属性值
 
```
   // 基础类型检测, null意味着任何类型都行
  {propA: Number,}
  // 多种类型
```

### 数组做对象的属性值
```
 { propB: [String, Number],}
  // 必传且是String
  
```

```

class:['red','big']

```
  
### 对象做对象的属性值

```
 { 
 propC: {
   type: String,
   required: true
  },
 }
  // 数组、默认值是一个工厂函数返回对象
  
```

  
### 动态属性:函数做对象的动态属性值

```
  propE: {

   default: function() {

    return { message: "I am from propE." };
   }
  },

```
```
computed:{

com(){

return 1

}

}


```
