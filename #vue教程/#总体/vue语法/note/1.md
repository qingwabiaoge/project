# js对象的属性值

### 基本类型值做属性值

```javascript
   // 基础类型检测, null意味着任何类型都行
  {propA: Number,}
  // 多种类型
```

### 数组做属性值

```json
 { propB: [String, Number],}
  // 必传且是String
  
```

### 对象做属性值

```
 { 
 propC: {
   type: String,
   required: true
  },
 }
  // 数组、默认值是一个工厂函数返回对象
  
```

### 函数做属性值 (动态属性)

```
  props: {

   id: function() {

    return this.$route.params.id};
   }
  },

```
