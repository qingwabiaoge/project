
# 指令写法

```html
<input v-bind="{color:red}">
<input v-bind:class="xxx">
<template v-slot:name="xxx">xxx</template>
<input v-model.trim="msg">
<button v-on:onlick='fn'></button>
```

```html
{{}}

{{msg|myFilter(n)}}
```

# 指令和js对象的属性值

### 基本类型值

```html
<h1 v-bind:class="['red','big']"></h1>
```

```html
   // 基础类型检测, null意味着任何类型都行
  {propA: Number,}
  // 多种类型
```

### 数组

```html
<h1 v-bind:class="['red','big']"></h1>
```

```json
 { propB: [String, Number],}
  // 必传且是String
  
```

### 对象

```
<h1 v-bind:class="{red:true,big:true}"></h1>
```

```
 { 
 propC: {
   type: String,
   required: true
  },
 }
  // 数组、默认值是一个工厂函数返回对象
  
```
