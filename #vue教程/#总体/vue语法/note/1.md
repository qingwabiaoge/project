
# 模板写法

```html
<input v-bind:class="xxx">

<input v-bind="{color:red}">

<template v-slot:name="xxx">xxx</template>

```


# 对象的属性值

### 基本类型值
 
```
   // 基础类型检测, null意味着任何类型都行
  {propA: Number,}
  // 多种类型
```

### 数组
```
 { propB: [String, Number],}
  // 必传且是String
  
```

```

<h1 :class="['red','big']"></h1>

```
  
### 对象

```
 { 
 propC: {
   type: String,
   required: true
  },
 }
  // 数组、默认值是一个工厂函数返回对象
  
```
