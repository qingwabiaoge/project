# 条件
vue

```
< div v-if='length>5'>1</div>
< div v-elseif='length>2'>2</div>
< div v-else>3</div>
```
小程序

```
<view wx:if="{{length > 5}}"> 1 </view>
<view wx:elif="{{length > 2}}"> 2 </view>
<view wx:else> 3 </view>
```
原生

```
if(length>5){}
elseif(length>2){}
else{}
```
